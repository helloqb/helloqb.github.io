四宫格办公壁纸制作
2020-06-03 00:00 folder [Photoshop]  

介绍

以 Windows 自带纯色壁纸为基础，效果如下：

流程

1. 用 Photoshop 打开壁纸

个人喜欢纯色壁纸，Windows10 上可通过背景设置为纯色壁纸后截屏得到。

2. 新建图层并填充为白色

操作： 图层（菜单栏）-> 新建 -> 图层 -> 编辑（菜单栏）-> 填充 -> 背景色

3. 缩小一半长宽比

操作：编辑（菜单栏）-> 自由变换 -> W:49% H:49%

49% 也就是留了 2% 的缝隙，可根据自己喜好修改。

4. 切割图层

以变换后的图层作为选区，分别左上、左下、右上、右下复制图层。

操作：

  1. 对齐：选中两个图层，点击菜单栏图层里的对齐，左对齐，然后再顶部对齐；

  1. 载入选区：按住 Ctrl，并点击白色图层的缩略图；
  2. 复制图层：点击背景图层，点击 Ctrl + J，将所选区域复制到新图层中；

  1. 同样方式左下、右上、右下生成另外三个图层；

5. 增加图层样式

操作：

  1. 双击图层打开图层样式，选择投影，距离和大小设置为 20 像素；
  2. 右键图层拷贝图层样式，然后复制到另外三个图层中；

大功告成，当然可根据自己喜好选择样式，添加相关文字等。
