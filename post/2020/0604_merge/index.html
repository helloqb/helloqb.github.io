EXCEL 技巧之 - 相同单元格合并
2020-06-04 00:00 folder [EXCEL]  

需求

需求其实很简单，就是合并单元格，行数少的情况下一个一个点就行了，但是行数多的情况下就麻烦了，秉承着重复性工作一定能被计算机代替的想法，翻阅很久找到了方法，在此分享。

实现

1. 数据（菜单栏） -> 分类汇总

其中：

  • 分类字段：需要合并的列，本例中为身份；
  • 汇总方式：无关，默认即可；
  • 选定汇总项：同分类字段；
  • 替换当前分类汇总：无关，默认即可；
  • 每组数据分页：不选；
  • 汇总结果显示在数据下方：选中

2. 汇总列合并空值单元格

对生成的汇总列进行空值定位，并合并空值单元格。操作如下：

汇总列第一行填 1 -> 选中汇总列 -> 开始(菜单栏) -> 查找和选择 -> 定位条件：空值 -> 合并后居中

3. 间隔一行（非必须）

如果想要每一类间隔一行的话，使用操作 2 的步骤定位合并列（定位条件：数值）然后插入工作表行就行了。

提一下这个是觉得定位这个功能很强大的样子，类似于编辑器中的多光标选择。

4. 格式刷合并列

将汇总列格式刷到合并列。

4. 删除分类汇总

数据（菜单栏） -> 分类汇总 -> 全部删除。
