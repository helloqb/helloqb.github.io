  • Mac 如何快速启动 Aria2

Aria2

官网 对其的定义：

aria2 是一个轻量级的多协议和多源命令行 下载工具。它支持 HTTP/HTTPS、FTP、SFTP、 BitTorrent 和 Metalink。aria2 可以通过内置的 JSON-RPC 和 XML-RPC接口进行操作。

用到的插件：

  • 可视化窗口：AriaNg
  • Chrome 插件：Camtd - Aria2下载管理器
  • Chrome 插件（推荐）：Aria2 for Chrome

配置

新建目录结构：

├── Aria2
│   ├── aria2.conf
│   ├── aria2.session
│   └── downloads

其中 aria2.session 为空文件，用来重新加载任务；downloads 为下载存放目录；aria2.conf 为参数配置文件，配置如下：

### Basic ###
dir=./downloads
input-file=./aria2.session
save-session=./aria2.session
save-session-interval=60
continue=true

### Adcanced ###
disk-cache=64M
file-allocation=none
# daemon=true
quiet=true

### RPC ###
enable-rpc=true
rpc-allow-origin-all=true
rpc-listen-all=true

### HTTP/FTP ###
max-connection-per-server=16
min-split-size=8M
split=32
user-agent=Transmission/2.77

对于每个参数的含义，可以下载 AriaNg，上面有完整的描述。

Mac 启动

推荐 brew 方式安装。

brew install aria2

启动命令：

# 先进入 Aria2 目录
aria2c --conf-path=aria2.conf

Windows 启动

官网 直接下载 exe 文件放到同目录下，启动命令，

# 先进入 Aria2 目录
aria2c.exe --conf-path=aria2.conf

如何实现开机自启

首先编写脚本 HideRun.vbs 实现隐藏窗口启动，

CreateObject("WScript.Shell").Run "aria2c.exe --conf-path=aria2.conf",0

然后在 用户\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup 目录中创建此脚本的快捷方式即可。
