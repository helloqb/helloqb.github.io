Smartisan TNT 连接遥控器变成超强机顶盒
连接显示器就变成大屏操作系统的 Smartisan TNT OS，感觉很适合用作机顶盒。
2022-01-13 00:00 folder [Android]   label Smartisan  

前言

家里的机顶盒是电信的，基本全部限制死了，无法安装第三方 APP，更别说看 Netflix 了，本来想着闲鱼淘一个二手破解的机顶盒玩玩，突然想到闲置的坚果 R1 及其备受争议的 TNT 大屏操作系统，好像还挺适合当机顶盒用的。

遥控

其实本可以没有这篇文章，因为 TNT 系统已经够成熟了，连上键鼠完全可以像电脑一样操作，当作机顶盒播放视频绰绰有余。但是客厅里摆一套键鼠总觉得不够优雅，所以想着可不可以实现像遥控器操控机顶盒那样。

手机遥控

第一个想法就是，如果有一个软件，可以将一个手机作为另外一个手机的输入设备，那么其实就可以摆脱键鼠了。

Android 9.0 + 蓝牙 HID

Android 9.0 自带将手机作为输入设备的功能，遥控助手 和 妙妙触控 都是利用此原理，但是遗憾的是国内大部分手机都阉割蓝牙 HID 功能，像我手上的 OPPO 就不行，用了别人的荣耀手机，可以实现，但是触控容易断，体验较差。

局域网遥控

KDE Connect 利用局域网实现手机对手机之间遥控，体验不错，较为灵敏，但遗憾的是不兼容 TNT，即使在连接显示器的情况下，也是只能操作手机。期待以后兼容（希望太小了……）

官方软件遥控

国内大部分视频 APP 都会有个投屏功能，手机上找到想看的视频，点击投屏就可以投到已安装相应 TV 版的盒子上了。

目前来说，B 站的体验较好，有一个全局投屏的功能，一开始连接上之后，后面每个视频都会自动在 TV 上播放。

遥控器

软件层面的遥控试了一圈都不行，本来想着买个迷你版的键鼠，逛了一下，竟然找到一个支持鼠标功能的遥控器。

官方明确说支持 Android 手机，支持蓝牙配对，买回来试了下，完美支持，甚至支持 TNT 的语音功能，但是得是在插入键盘的情况下😳，可能插入键盘系统才会作映射吧。

软件体验

Kodi

Kodi 是一款开源媒体播放器，有庞大的附加组件社区，是机顶盒的热门软件。支持本地所有类型的媒体，支持 DLNA/SMB 远程播放，支持 IPTV 电视流播放。国外主流的视频网站插件 Netfilx、Youtube 都有，国内的优酷、B 站也有，只是更新维护不够。

中文设置

支持中文，但是直接设置会乱码，需要先修改字体；

  1. Settings -> Interface -> Skin -> Fonts，设置为 Aria based；

  1. Settings -> Interface -> Regional -> Language，设置为 Chinese (Simple)；

手机远程遥控

官方的遥控软件 Kore 体验也非常好，Android 和 IOS 都支持，而且很多第三方的遥控软件也都会支持 Kodi，以 Android 为例：

  1. 设置 -> 服务 -> 控制；允许通过 HTTP 远程控制，端口号建议改成非常用端口如 18080；

  1. Google Play 下载 Kore，需要在同一局域网下，软件会自动识别出设备连接；

DLNA 播放

首先需要设置里打开此功能：

  1. 设置 -> 服务 -> UPnP/DLNA，启用 UPnP 支持；

  1. 电影 -> 添加视频源 -> UPnp 设备；局域网内的设备就会自动展示出来；

播放电视

首先需要下载 IPTV 客户端插件，插件 -> PVR 客户端 -> PVR IPTV Simple Client；

播放电视首先需要电视源，GitHub 上有很多这类项目，我用的是这个 链接，国外源也可以播放，但是也听不懂，所以就选择中文分类的列表导入。

在插件中，设置选择远程路径，输入 https://iptv-org.github.io/iptv/languages/zho.m3u。

等待加载片刻，电视频道里就会出现查询到的频道里。如果加载不出来，可能是 GitHub 国内访问太慢，可以将此文件下载到手机上，然后在插件中设置本地路径，导入即可。

播放 Netflix

Kodi 官方库中没有此插件，可能是版权的原因吧。但是民间自有大神，名为 Stefano Gottardo 的开发人员开发了 Netflix 插件，测试可用。安装步骤稍微复杂些：

  1. python 脚本生成 Key；
  2. 下载第三方源插件，选择插件从 zip 文件中导入；
  3. 选择从第三方库安装 Netflix 插件；
  4. 点击 Netflix，使用身份密钥登陆；

YouTube

YouTube 使用有三种方法：

1. 直接安装 APP

其实 TNT 系统已经兼容 YouTube 了，打开会展开为平板模式，但是无法通过遥控器进行按键操作，所以作为次要选择。

2. Kobi 安装 插件

另外想到的方案，就是 Kobi 插件了，这种方案肯定可行，而且体验也会不错，毕竟融入到 Kobi 平台里了。

3. SmartTube Next

最后想到的是 YouTube TV，既然是 TV 版本，肯定会兼容遥控器，遗憾的是官方只支持盒子安装，无法在手机上安装。

网上搜了下，竟然找到了一个代替 YouTube TV 的第三方软件 SmartTube Next。兼容遥控器，使用起来体验很好，唯一有点不好的是每进入下一个页面，系统就会生成一个独立的窗口。

其他体验好的软件

支持遥控器按钮操作：

  • MX Player

适配 TNT 全屏：

  • YouTube Kids
  • Google Drive
  • 哔哩哔哩等各大视频网站的 TV 版
