pub&ppk

SSH 客户端 Putty(.ppk) 和 OpenSSH(.pub) 相互转换：

  1. Git 中输入 ssh-keygen 生成密钥到 .ssh 目录中，其中 id_rsa 为私钥，id_rsa.pub 为公钥，文本打开 id_rsa.pub，复制到云服务器上即可。
  2. 若想用 Putty 加载（SourceTree默认），则 SourceTree 中工具->选项->SSH客户端配置改为 Putty/Plink ，然后返回上一页打开 Putty Key Generator -> Conversions -> Import Key(导入id_rsa)， Save private key， 生成了ppk文件，导入即可
  3. 反过来打开 Putty Key Generator -> Generate -> Save private key，然后导入， 同时页面上的Key复制到云服务器上.

参考： https://git-scm.com/book/en/v2/Git-on-the-Server-Generating-Your-SSH-Public-Key https://blog.csdn.net/caoshiying/article/details/78041123 https://www.oschina.net/question/2272241_2136680?sort=time

本地项目到远程仓库

步骤：

  1. git init 命令将项目变成仓库
  2. 远程创建空的仓库
  3. 将本地连接到远程 git remote add origin url git push -u origin master

注： 远程创建的为非空仓库（会有README.txt）则需要 git pull --rebase origin master，之后再 push。
